Setting up BlueStacks and Windows for the WOTV Bot

The WOTV Bot macro tool uses image recognition to make decisions in the game. It needs to have a clear view of the screen and the resolution has to be a specific size for it to make confident decisions.

Here are some tips to help ensure it works with your BlueStacks setup.

Close sidebars and other windows

When the bot is running the BlueStacks window should look like this:

Windows display scaled to 100%

Ensure that your Windows display settings are scaled to 100%:

Pin BlueStacks to top

Prevent other windows from obscuring BlueStacks by using the pin to top option. This isn’t necessary, but if another windows is dragged over the emulator then it will not be able to proceed and in rare cases may even click on the window that’s covering it.

Set BlueStacks resolution

If the bot is having trouble resizing the emulator to the correct dimensions, try changing the emulator’s display settings to match the ones below. Additionally, you can try closing BlueStacks completely, making sure it exits fully by clsoing the window and clicking exit from the taskbar.
